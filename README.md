# オートポイエーシス可能性：システムが生む生命らしさの定量的評価

## 概要

この研究では，F．J．Varela らが提唱した Autopoiesis という概念を用いて，システムにおける「生命らしさ」について考える．

Autopoiesis　として満たすべき条件を数学的に整理し，システムがオートポイエーシス単位体を産出する能力として “オートポイエーシス可能性” を新たに定義し，
システムのルールと観察ルールと初期条件から導かれるオートポイエーシス可能性を下限的に求める手法を提案する．

提案手法の確認と結果的事実として，計算論的エージェントモデルの１つである Particle Lenia を題材に，
複数の実験条件におけるオートポイエーシス可能性を評価し，条件の比較に基づいた生命らしさの変化についての説明と考察を試みる.
